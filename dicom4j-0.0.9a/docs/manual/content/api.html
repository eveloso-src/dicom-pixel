API Overview

Dicom4j's API contains 3 differents parts, which cover different need related to dicom. The first one is the core API, which offer way to manage dicom in java. The second is a toolkit, it purpose is to provide classes to help a dicom developper. The last part is a plug-in architecture which offers out of the box component for end-user applications.
